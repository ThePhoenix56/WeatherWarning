# WeatherWarning
A mobile app written in React Native to serve as a notification tool for any upcoming weather alerts from the Swedish Meteorological and Hydrological Institute (SMHI).

## Development Status
Still in development! See issues for any upcoming features.

## Installation
The app is neither listed on the Google Play Store or the iOS App Store. As it's not free to list apps there, I unfortunately don't plan on listing this over there for now.
